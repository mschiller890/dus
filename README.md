# dus

A simple terminal tool that shows disk usage in a clean, visual way.

It groups usage by physical disk and displays it with easy-to-read bars and colors.

## Why?
Most disk tools show every partition separately, which can get messy.  
`dus` just shows what’s actually going on per drive.
